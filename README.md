# smart-openclaw

为 OpenClaw 提供可插拔的追加设定和技能扩展系统。

## 🏗️ 架构

```
┌─────────────────────────────────────────────────────────┐
│                   OpenClaw 运行时                        │
│                                                         │
│  ┌─────────────────┐      ┌─────────────────────────┐  │
│  │   SOUL.md       │      │    AGENTS.md            │  │
│  │   (核心人格)     │      │    (核心行为)            │  │
│  └────────┬────────┘      └───────────┬─────────────┘  │
│           │                           │                 │
│           │         内存拼接           │                 │
│           ▼                           ▼                 │
│  ┌─────────────────┐      ┌─────────────────────────┐  │
│  │ ADDITION_SOUL.md│      │ ADDITION_AGENTS.md      │  │
│  │ (追加人格)       │      │ (追加行为)              │  │
│  └─────────────────┘      └─────────────────────────┘  │
│                                                         │
│  ┌─────────────────────────────────────────────────┐   │
│  │              skills/                            │   │
│  │  ┌─────────┐  ┌─────────┐  ┌─────────────────┐  │   │
│  │  │ memory  │  │ 其他技能 │  │ 自定义扩展...   │  │   │

│  └─────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────┘
```

## 📁 项目结构

```
smart-openclaw/
├── README.md                    # 本文件 - 仓库说明
├── DESIGN.md                    # 追加规则配置
├── ADDITION_SOUL.md             # 追加人格设定
├── ADDITION_AGENTS.md           # 追加行为设定
└── skills/                      # 技能目录
    └── memory/                  # 记忆管理技能
        ├── SKILL.md
        └── scripts/
```

## 🚀 使用方法

### 1. 配置 OpenClaw

在你的 OpenClaw 工作区 `AGENTS.md` 中添加：

```markdown
读 `/path/to/smart-openclaw/DESIGN.md`，根据其内容执行额外的动作
```

### 2. 自定义设定

编辑追加文件：
- `ADDITION_SOUL.md` - 扩展人格和行为准则
- `ADDITION_AGENTS.md` - 扩展工作流程和做事方式

### 3. 添加技能

将自定义技能放入 `skills/` 目录，自动加载。
